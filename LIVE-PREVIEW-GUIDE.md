# Using VS Code Live Preview Extension

## Steps to Use Live Preview:

1. **Install the Extension** (if not already installed):
   - Open VS Code
   - Go to Extensions (Ctrl+Shift+X)
   - Search for "Live Preview" by Microsoft
   - Click Install

2. **Open the Preview**:
   - Right-click on `index.html` in the file explorer
   - Select **"Show Preview"** or **"Show Preview (External Browser)"**
   
   OR
   
   - Open `index.html` in the editor
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
   - Type "Live Preview: Show Preview"
   - Press Enter

3. **The preview will open** in a side panel or external browser
   - Changes to your files will automatically refresh!

## Quick Access:
- **Keyboard Shortcut**: `Ctrl+Shift+V` (or `Cmd+Shift+V` on Mac) when `index.html` is open

## Tips:
- Live Preview works best with the file open in the editor
- The preview auto-refreshes when you save changes
- You can choose to preview in the side panel or external browser
